# Simple Interest Calculator

A simple Bash script to calculate simple interest based on principal amount, rate of interest, and time period.

## Description
This project contains a bash script that calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

text

## Features
- Takes user input for principal amount
- Takes user input for rate of interest
- Takes user input for time period
- Calculates and displays the simple interest

## Usage
1. Make the script executable:
   ```bash
   chmod +x simple-interest.sh
Run the script:

bash
```
./simple-interest.sh
```
Follow the prompts to enter:

Principal amount

Rate of interest

Time period (in years)

Example
text
Enter the principal amount: 1000
Enter the rate of interest: 5
Enter the time period (in years): 2
Simple Interest = 100.0
